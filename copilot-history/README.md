# Copilot History

This folder tracks every major change made by AI agents (GitHub Copilot, etc.) to the Mitroo project. Each entry documents **what was done**, **files affected**, and **instructions for the next commit**.

## Structure

| File | Purpose |
|------|---------|
| `CHANGELOG.md` | Chronological log of all agent sessions |
| `NEXT_STEPS.md` | Prioritised backlog / instructions for the next agent session |
| `ARCHITECTURE.md` | High-level architecture reference |
